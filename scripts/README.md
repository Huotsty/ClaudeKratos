# Kratos Railway Deployment Scripts

This directory contains helper scripts for Railway deployment.

## generate-secret.sh

Generates a secure random secret for Kratos:

```bash
bash scripts/generate-secret.sh
```

## check-health.sh

Checks if Kratos is healthy:

```bash
bash scripts/check-health.sh https://your-kratos-domain.railway.app
```

## test-smtp.sh

Tests SMTP configuration:

```bash
bash scripts/test-smtp.sh
```
